# Getting Started

Welcome to the Climate & Health Analytics Platform (CHAP). CHAP is an open platform for climate-informed disease forecasting. It connects researchers and epidemiological models to health practitioners, integrating directly with DHIS2â€”the world's leading health information system.

## Platform Components

- **Chap Core**: Python engine for model orchestration, data pipelines, validation, and hyperparameter tuning
- **Modeling App**: User interface for training models, visualizing predictions, and evaluating accuracy
- **Climate App**: Import weather and environmental data from sources like ERA5-Land
